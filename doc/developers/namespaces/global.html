      Returns the code that embeds an EJS applet into Moodle

      generate_applet_embedding_code(\stdClass $ejsapp, int | null $sarlabinstance, int | null $practiceid, string | null $state_file, int | null $col_session, string | null $col_ip, int | null $col_port, int | null $session_director) : string

      This function returns the HTML and JavaScript code that embeds an EJS applet into Moodle It is used for three different cases: 1) when only the EJSApp activity is being used 2) when the EJSApp File Browser is used to load a state file 3) when the EJSApp Collab Session is used

      Parameters

      $ejsapp

      \stdClass

      record from table ejsapp

      $sarlabinstance

      intnull

      sarlab id (null if sarlab is not used)

      $practiceid

      intnull

      practice id (null if sarlab is not used)

      $state_file

      stringnull

      if generate_applet_embedding_code is called from block ejsapp_file_browser it is the name of the xml file that stores the state of an EJS applet, elsewhere it is null

      $col_session

      intnull

      if generate_applet_embedding_code is called from block ejsapp_collab_sessioncollaborative it is the session id, elsewhere it is null

      $col_ip

      stringnull

      if generate_applet_embedding_code is called from block ejsapp_collab_sessioncollaborative it is the IP of the collaborative session master user, elsewhere it is null

      $col_port

      intnull

      if generate_applet_embedding_code is called from block ejsapp_collab_sessioncollaborative it is the port of the collaborative session master user, elsewhere it is null

      $session_director

      intnull

      if generate_applet_embedding_code is called from block ejsapp_collab_sessioncollaborative it is the id of the collaborative session master user, elsewhere it is null

      Returns

      stringcode that embeds an EJS applet into Moodle
